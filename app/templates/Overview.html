Project Overview

What does the project do?

 Our group created a model that can generate rap music based on a human input. The model will replicate the style of a certain artist, either Eminem, Tyler the Creator, or Tupac, in order to create a cohesive rap song. 

How does the project work?

  

The group trained the model using datasets from kaggle to teach the artificial intelligence about each artists style of rap. With all of the data gathered, the computer can generate music of a simialr style.  

Why a rap generator?

Our group chose to make a rap generator. In one aspect, the generator can be used for fun; people can quickly write their own rap music with friends to sing. On the other hand, artists who are in a creative block could use this project to generate lyrics if they are out of ideas. 

Tools Used

Kaggle

Kaggle provided several datatsets that were used to train each model to their respective artist. This data was then pre-processed to remove profanity.

Jupyter Notebooks

Jupyter Notebooks were used to learn the fundamentals of python with respect to artifical intellignece, such as NLTK. 

GPT Model

The GPT model makes it possible to generate text that has a similar style to text found in the data. This makes it possible to generate lyrics that resemble certain artists.
